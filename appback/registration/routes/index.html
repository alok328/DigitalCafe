/register

This is the endpoint used to register new students in the database.

The endpoint takes the following params in its body:

  • First Name
  • Last Name
  • Email
  • Roll
  • Hostel
  • Password

The endpoint has the following response codes:

  • 201 - Successful registration
  • 409 - User already registered
  • 500 - Server Error


/login

This is the endpoint used to authenticate users before giving access to the data.

The endpoint takes the following params in its body:

  • Roll
  • Password

The endpoint has the following response codes:

  • 200 - Successful login with ACCESS_TOKEN
  • 401 - Invalid credentials
  • 404 - User not registered

https://digitalcafe.herokuapp.com/user/{roll}


The following routes require the ACCESS_TOKEN in the header when making request

/balance

This is the endpoint used to get user balance for given roll.

The endpoint has the following response codes:

  • 200 - Balance retrieved
  • 403 - Invalid ACCESS_TOKEN
  • 404 - User not registered


/transaction

This is the endpoint used to get user transactions for given roll.

The endpoint has the following response codes:

  • 200 - List of user transactions
  • 403 - Invalid ACCESS_TOKEN
  • 404 - User not registered


/transaction

This is the endpoint used to add user transactions for given roll.

The endpoint takes the following params in its body:

  • Roll
  • Amount
  • Menu for which the amount is to be deducted
  • Hostel

The endpoint has the following response codes:

  • 201 - Transaction added to user's account
  • 400 - Insufficient balance
  • 401 - User is from a different hostel than the one QR was scanned from
  • 403 - Invalid ACCESS_TOKEN
  • 404 - User not registered
